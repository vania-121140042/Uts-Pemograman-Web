Apa itu Website?

Website merupakan halaman yang berisi informasi yang tersedia melalui jalur internet, sehingga dapat diakses oleh seleuruh pengguna di dunia dimanapun dan kapanpun. Website terdiri dari beberapa komponen teks, gambar, video, animasi, suara yang membuat website tersebut menarik.

Website pertama di dunia diluncurkan pada tahun 1991 oleh ahli komputer asal Inggris bernama Sir Timothy John “Tim” Berners-Lee. Awalnya, tujuan Tim dalam merancang website adalah untuk memfasilitasi pertukaran dan pembaruan informasi antar sesama peneliti di tempat kerjanya. Itu karena, website tersebut hanya dibangun menggunakan HTML (HyperText Markup Language). HTML adalah bahasa markup yang hanya mampu menampilkan teks dan gambar yang statis. Baru setelahnya pada tahun 1996, muncul inovasi bernama CSS (Cascading Style Sheets). CSS adalah teknologi yang diperkenalkan untuk mempercantik tampilan website.

Fungsi Website

Ada beberapa fungsi website yang perlu diketahui yaitu:

  • Sarana informasi
  • Membangun Brand Bisnis
  • Menyediakan Informasi produk dan Jasa
  • Meingkatkan Keterlibatan pelanggan
  • Meningkatkan kepercayaan pelanggan
